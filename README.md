# Bsit Exercise
## Description
This project has been developped for BSit interview.  
The goal is to develop an API serving data from a given dataset and display them on a webpage.  
## Getting started
Once you cloned the project, first of all run `npm install`  
Here are the different commands you can run :  
 - `npm start` will build sources and run the server
 - `npm run dev` will start the project with listener to auto-refresh during developpment  
 
You can now browse the project on http://localhost:3000
